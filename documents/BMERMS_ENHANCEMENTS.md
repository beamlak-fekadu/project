# BMERMS Enhancements

This branch (`bmerms-enhancements`) tracks planned improvements to the Biomedical Equipment and Resource Management System (BMERMS/MEMIS).

## Scope

- UI/UX improvements for equipment tracking workflows
- Enhanced reporting and dashboard analytics
- Role-based access control refinements
- Performance optimizations for large equipment inventories

## Notes

- Built on top of the existing Next.js + Supabase stack
- Follows the existing service/component architecture in `src/`
- Database changes should include corresponding migration files under `supabase/migrations/`
